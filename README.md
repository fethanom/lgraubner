> 改名为 qox，重新开坑，在 react 中使用 composition API

### Use

```shell
npm i qox -S
```

```js
import { setup, reactive } from 'qox'
import { render } from 'react-dom'

const App = setup(() => {
  const data = reactive({ msg: 'hello world' })
  return () => <div>{data.foo}</div>
})
render(<App />, document.body)
```
